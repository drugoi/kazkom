Информация о командах и их возможностях Варианты команд: HELP, H, HLP Порядок ввода: Ввод одного из вариантов команды, пробел, ввод наименования команды Пример вводa: H [наименование команды]
Информация о фактическом остатке на текущем счете Варианты команд: B, BAL, BALANCE Порядок ввода: Для просмотра остатков всех подвязанных счетов достаточно ввести в тексте sms один из вариантов команды. Для просмотра остатка по конкретному счету нужно ввести один из вариантов команды пробел, ввод маскированного кода текущего счета Пример вводa: B [маскированный код счета]
Возможность смены языка информационных sms-сообщений Варианты команд: LANGUAGE, LANG, L Порядок ввода: Ввод одного из вариантов команды, пробел, ввод языка RU-русский, RUTR-русский транслит, EN-английский, KKTR-казахский транслит Пример вводa: L [RU/RUTR/EN/KKTR]
Информация о движении по текущему счету (последние 3 операции) Варианты команд: T, TRA, TRANS, TRANSACT Порядок ввода: Ввод одного из вариантов команды, пробел, ввод маскированного кода текущего счета Пример вводa: Т [маскированный код счета]
Отправка платежей Варианты команд: P, PAY, PAYM, PAYMENT Порядок ввода: Ввод одного из вариантов команды, пробел,названия шаблона платежа, пробел, сумма платежа ( в случае указания суммы в шаблоне платежа, в sms можно не указывать) Пример вводa: P [название шаблона платежа] [сумма]
Возобновление sms-уведомлений Варианты команд: W, WAKE, WAKEUP Порядок ввода: Ввод одного из вариантов команды Пример вводa: W
Приостановление sms-уведомлений Варианты команд: S, SL ,SLEEP Порядок ввода: Ввод одного из вариантов команды Пример вводa: S
Команда GO на короткий номер 4343 предназначена для получения sms-сообщения с адресной ссылкой mbank.kz Варианты команд: GO, G Порядок ввода: Ввод одного из вариантов команды Пример вводa: G
Включение автоплатежа Варианты команд: AUTO, A Порядок ввода: AUTO [название шаблона платежа] ON Пример вводa: Ввод одной из вариантов команды, пробел, название шаблона платежа, пробел, ON

Подключение системы"Mobilebank" (без учета НДС)

Ежемесячная абонентская плата за ведение счетов электронных клиентов с использованием системы "Mobilebank"

Взимается дополнительно к стандартной комиссии за ведение счетов.

Комиссия за проведение запросов, переводов, уведомлений и предоставление выписки по банковскому счету на основании SMS-сообщений.

Взимается по каждой операции/представленной выписке дополнительно к стандартным комиссиям за соответствующую услугу.

Преимущества
Осуществление переводов денежных средств в тенге:
  • Между клиентами КАЗКОМа;
  • Между своими текущими счетами, открытыми в КАЗКОМе;
  • На банковские счета, открытые в других банках (собственные и принадлежащие третьим лицам).
Информационные услуги:
  • Остатки по текущим счетам в КАЗКОМе;
  • SMS – уведомления об изменении остатков по счетам;
  • Последние операции по своим текущим счетам;
  • Информация о расположении банкоматов и операционных отделений КАЗКОМа.
